Changes to Play Data Transfer

Date 2017-04-26 三 Tags POST
Thank you for playing School Idol Festival. From mid May, we plan to change the way you transfer your game data from one device to another.

What’s being changed?
You will be required to enter your Transfer ID and Transfer Passcode to transfer your game data.

When will it change?
Mid May (TBC)

We will start issuing Transfer IDs to those who already have valid Transfer Passcodes. This Transfer ID will be required to transfer game data after the update, so please make sure to take a screen shot and save it or write it down for safe keeping.

・If you do not have a Transfer Passcode already issued, or it has expired, please issue a new Transfer Passcode and you will be sent a Transfer ID.
・Your Transfer ID will be shown upon boot-up.
・Your Transfer ID will be sent to you within 48 hours of confirming the valid code.

You can continue to use Game Center to back up your game data. Please consider linking your game to Game Center if you plan to transfer game data to a device using the same OS.

Notes
- If the Game Center Sign In dialog pops up, please use your Apple ID to sign in.
- You will be required to sign in on your new device using the same Apple ID you linked with on the old device.
- You will not be able to transfer game data from an iOS device to an Android device using Game Center.
- Transferring game data using a Transfer Passcode will automatically unlink your Game Center account.
